# Rubik's Cube Resources

## Knowledge

- [Ruwix — How to Solve the Rubik's Cube (Beginner Method)](https://ruwix.com/the-rubiks-cube/how-to-solve-the-rubiks-cube-beginners-method/)
  Clear, visual, step-by-step beginner guide with diagrams. Use for: all seven steps of the layer-by-layer method and algorithm reference.

- [SolveTheCube — Beginner's Guide](https://solvethecube.com/)
  Clean, modern beginner guide with notation overview. Use for: notation explanations, first and second layer techniques, algorithm diagrams.

- [CubeSkills — Beginner's Method PDF (Feliks Zemdegs)](https://www.cubeskills.com/uploads/pdf/tutorials/the-beginners-method-for-solving-the-rubiks-cube.pdf)
  Written by world record holder Feliks Zemdegs. Use for: authoritative beginner algorithms in standard notation, especially the last layer.

- [YouCanDoTheCube — Official Solution Guide (PDF)](https://assets.ctfassets.net/r3qu44etwf9a/6kAQCoLmbXXu29TTuArrk1/404118e1f9bfb6f9997157a284bbc572/Rubiks_Solution-Guide_3x3.pdf)
  Official Rubik's Brand guide. Use for: overview of the full official seven-step beginner method.

- [SpeedCube.us — How to Solve a Rubik's Cube](https://www.speedcube.us/pages/how-to-solve-a-rubiks-cube)
  Another beginner guide with video support. Use for: visual learners who benefit from seeing the moves.

## Wisdom (Communities)

- [r/Cubers](https://www.reddit.com/r/Cubers/)
  Friendly subreddit welcoming all skill levels. Good for encouragement, cube recommendations, and beginner questions.

- [SpeedSolving.com Forums](https://www.speedsolving.com/forum/)
  More technical discussion. Better suited once the learner reaches intermediate level.

## Gaps

- No resource found specifically aimed at ages 6–9. Existing guides are technically correct but written for older audiences — this is why lessons are simplified and rebuilt for the 8-year-old learner.
